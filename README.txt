pigpio_builder -- Automatic downloading and builing the latest version of pigpio with apt upgrade
============== == ========= =========== === ======= === ====== ======= == ====== ==== === =======

Just a simple tool chekcing for a new version and update pigpio_builder from the server

This program automatically builds the latest release of pigpio
Options:
help, -h or --help: print this help file and exit
force             : force update, even if no new version is available
update            : only prints message when an update is available
upgrade           : installs update when available
<no arguments>    : installs update when available (more verbose)

That's all for now ...
